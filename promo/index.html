"WordPress-Easy" Installation

You don't need a degree in server administration to self-host. DIY Inventory was built to be extracted and set up in less than 5 minutes.

1

Download ZIP

Grab the latest pre-compiled, production-ready release ZIP containing the optimized application build directly from GitHub.

$ wget [Dynamic Release URL]
2

Upload & Extract

Upload the ZIP archive to your web host server folder (e.g., standard Apache, Nginx, Raspberry Pi, cPanel, or local XAMPP).

$ unzip diy-inventory.zip
3

Run Web Setup

Open your browser and load the URL. The automated 5-minute setup wizard takes over, validates server packages, links MySQL, and secures your password.
